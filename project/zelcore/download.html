Features

ZelCore v3.8.0 changes

Features

  • Service for Staking: Automatically activate ZelCore+ for users holding at least 5000 ZEL in light accounts
  • It is now possible to attach custom InvoiceID on Ripple network
  • Bitpay Json Payment Protocol scheme has been integrated. This allows smooth payment for invoices in many currencies such as BTC, ETH, XRP or BCH. Simply click on invoice link and proceed with payments in ZelCore
  • Further notification of d2FA not enabled
  • Shows sum of ZelNode rewards on over in ZelNode App
  • It is now possible to force a custom Nonce on Ethereum network. Use with caution
  • Following Full Node binaries updated: ZEL, ZEC, BTC, RVN, BTX, BTG, XSG, KMD, ZCOIN, BZE
  • ZelCore now has more automated fallbacks to other backend solutions in case a preselected one is not responding properly
  • Following Ethereum Tokens are now natively available: WHALE, WBTC
  • More clear text descriptions on account creation
  • Kadena now supports 20 chains

Bug fixes

  • UTXO fee calculation slightly corrected
  • Temporarily disables hiding ZelCore to Windows Tray due to bug investigation
