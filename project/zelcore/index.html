About zelcore

ZelCore is a multi asset platform and wallet, free-to-use by all, with top quick-swap exchanges. ZelCore+ unlocks advanced trading functionality with API integrations to the top exchanges + TradingView.

Features

Support for multiple accounts

Our login system allows you to have multiple encrypted wallets and accounts on the same device and access the same account on multiple devices while maintaining complete privacy as we don't store or transmit any user personal data. Privacy is something Zel strongly believe in.

Multi-Asset Encrypted Wallet

ZelCore keeps your assets safe. With support for 100+ assets including $ZEL, $BTC, $LTC, $ETH, $XMR, $ZEC, $USDT, and many more.

Built-in Exchanges

ZelCore integrates built-in quick swap exchanges directly inside of ZelCore Desktop & Mobile. But if you mean business and want to take on some real trading, check out ZelCore+ which offers full API integration to your favourite top exchanges like Binance, Bittrex, and Kraken.
